## 数据结构

### 优惠券类型
- 满减券 (FULL_REDUCTION) - 满X元减Y元
- 折扣券 (DISCOUNT) - 享受X折优惠
- 无门槛券 (NO_THRESHOLD) - 直接抵扣

### 优惠券状态
- 未开始 (NOT_STARTED)
- 进行中 (IN_PROGRESS)
- 已结束 (ENDED)
- 已停用 (DISABLED)

### 用户优惠券状态
- 未使用 (UNUSED)
- 已使用 (USED)
- 已过期 (EXPIRED)

## API接口清单

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| getAvailableCouponList | GET | /member/coupon/available-list | 获取可领取优惠券 |
| getCouponDetail | GET | /member/coupon/get | 获取优惠券详情 |
| receiveCoupon | POST | /member/coupon/receive | 领取优惠券 |
| getMyCouponList | GET | /member/coupon/my-list | 获取我的优惠券 |
| getOrderAvailableCoupons | POST | /member/coupon/order-available | 获取订单可用优惠券 |
| useCoupon | POST | /member/coupon/use | 使用优惠券 |
| getCouponUsageRecords | GET | /member/coupon/usage-records | 获取使用记录 |

## 页面功能说明

### 领券中心 (/coupon)
- 展示所有可领取的优惠券
- 按类型筛选（满减券/折扣券/无门槛券）
- 显示优惠券剩余数量和有效期
- 一键领取优惠券

### 我的优惠券 (/account/coupon)
- 标签页切换：未使用/已使用/已过期
- 显示优惠券详情（金额、有效期、使用条件）
- 即将过期提醒
- 跳转到题库使用优惠券
- 使用记录弹窗

### 订单支付集成 (/order/pay)
- 显示可用优惠券数量
- 优惠券选择弹窗
- 实时计算优惠后金额
- 不满足条件的优惠券置灰提示

## 关键决策

1. **适用类型设计** - 支持全部商品/指定分类/指定商品三种适用类型
2. **金额计算** - 后端计算实际优惠金额，前端展示
3. **过期提醒** - 3天内过期的优惠券显示警告提示
4. **领取限制** - 每人限领数量在优惠券层面控制
