## 技术方案

### 类型定义

```typescript
// types/search/index.d.ts

interface SearchReqVO {
  keyword: string
  type?: SearchType  // 'all' | 'qbank' | 'question' | 'article'
  pageNo?: number
  pageSize?: number
}

interface SearchResultItem {
  id: number
  type: SearchResultType
  title: string
  summary?: string
  categoryName?: string
  // ...
}
```

### API规划

```typescript
// api/search/index.ts

// 全局搜索
search: (params: SearchReqVO) => Promise<SearchRespVO>

// 搜索建议
getSuggestions: (keyword: string) => Promise<SearchSuggestRespVO>

// 热门搜索
getHotSearch: (limit: number) => Promise<HotSearchItem[]>

// 搜索历史（本地存储）
saveSearchHistory: (item: SearchHistoryItem) => void
getSearchHistory: () => SearchHistoryItem[]
clearSearchHistory: () => void
```

### 组件结构

```
GlobalSearch/
└── index.vue           # 全局搜索组件（导航栏使用）

pages/
└── search/
    └── index.vue       # 搜索结果页面
```
