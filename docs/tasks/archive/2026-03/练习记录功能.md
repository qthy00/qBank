## 需求描述

实现练习记录功能，支持用户查看历史练习记录，并可以继续未完成的练习。

### 业务背景
- 用户需要查看自己的练习历史
- 支持筛选和搜索练习记录
- 可以回到未完成的练习继续做题
- 展示练习的详细统计数据

### 功能目标
- [ ] 练习记录列表页面
- [ ] 支持按题库、练习类型筛选
- [ ] 支持继续未完成的练习
- [ ] 展示练习统计数据（正确率、用时、完成进度等）

### 约束条件
- 使用Element Plus组件
- 遵循项目设计规范
- 支持响应式布局

### 练习类型

| 类型 | 标识 | 说明 |
|------|------|------|
| 章节练习 | chapter | 按章节进行练习 |
| 顺序练习 | sequence | 按顺序做题 |
| 随机练习 | random | 随机抽题练习 |
| 每日练习 | daily | 每日推荐练习 |
| 模拟考试 | exam | 模拟真实考试 |

### 练习状态

| 状态 | 标识 | 说明 |
|------|------|------|
| 进行中 | in_progress | 已开始但未完成 |
| 已完成 | completed | 已完成全部题目 |
| 已放弃 | abandoned | 用户主动放弃 |
