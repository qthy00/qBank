## 需求描述

实现学习排行榜功能，展示用户的学习排名，包含三种榜单类型：做题数榜、正确率榜、学习时长榜。

### 业务背景
- 增强用户学习动力，通过排名竞争提升活跃度
- 展示社区学习氛围，促进用户参与
- 配合学习打卡功能，形成学习闭环

### 功能目标
- [x] 排行榜页面（三种榜单切换）
- [x] 用户排名展示（当前用户位置）
- [x] TOP10榜单展示
- [x] 榜单数据支持日/周/月时间维度
- [x] 支持查看用户学习详情

### 约束条件
- 使用Element Plus组件
- 遵循项目设计规范
- 支持响应式布局

## 方案设计

### 页面结构

```
pages/ranking/
├── index.vue              # 排行榜主页面
└── components/
    ├── RankingList.vue    # 榜单列表组件
    ├── RankingItem.vue    # 榜单单项组件
    └── UserRankCard.vue   # 当前用户排名卡片
```

### 榜单类型

| 榜单类型 | 标识 | 排序规则 |
|---------|------|---------|
| 做题数榜 | question_count | 做题数量降序 |
| 正确率榜 | accuracy_rate | 正确率降序（最低100题） |
| 学习时长榜 | study_duration | 学习时长降序 |

### 时间维度

| 维度 | 标识 | 说明 |
|-----|------|------|
| 日榜 | day | 今日数据 |
| 周榜 | week | 本周数据 |
| 月榜 | month | 本月数据 |
| 总榜 | total | 历史累计 |

## 技术实现

### 组件架构

```
RankingPage (pages/ranking/index.vue)
├── RankingList (components/RankingList.vue)
│   ├── RankingItem (components/RankingItem.vue) x 7
│   └── TOP3 Special Display
└── UserRankCard (components/UserRankCard.vue)
```

### 核心功能

1. **榜单切换**：三种榜单类型Tab切换，平滑过渡动画
2. **时间维度**：日/周/月/总榜切换，数据自动更新
3. **TOP3展示**：特殊的领奖台样式，金银铜区分
4. **趋势显示**：排名变化趋势，上升绿色、下降红色
5. **响应式布局**：PC端左右分栏，移动端上下排列

### 样式特点

- 使用项目CSS变量系统
- TOP3使用渐变色徽章（金/银/铜）
- 当前用户高亮显示
- 卡片阴影和悬停效果
- 平滑过渡动画
