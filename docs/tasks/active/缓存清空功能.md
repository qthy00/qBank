## 技术方案

### 需要清空的缓存类型

1. **localStorage**
   - token 相关数据
   - 用户偏好设置
   - 缓存的静态数据

2. **sessionStorage**
   - 临时会话数据
   - 页面状态缓存

3. **Pinia Store**
   - auth store 状态
   - user store 状态
   - 其他业务 store 状态
