## 方案设计

### 页面结构

```
/help                    # 帮助中心首页
/help/faq                # FAQ页面
/help/guide              # 使用指南列表
/help/guide/[id]         # 使用指南详情
/help/search             # 搜索结果页
/help/feedback           # 问题反馈页
```

### 功能模块

1. **帮助中心首页**
   - 搜索入口
   - 快捷分类入口（FAQ、使用指南、热门问题）
   - 常见问题推荐

2. **FAQ页面**
   - 分类导航
   - 问题列表（手风琴展开）
   - 搜索筛选

3. **使用指南**
   - 指南分类
   - 文章列表
   - 详情页（富文本展示）

4. **搜索功能**
   - 关键词搜索
   - 搜索结果分类展示
