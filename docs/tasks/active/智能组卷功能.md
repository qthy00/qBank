## 需求描述

实现智能组卷功能，根据用户的薄弱知识点自动组卷，进行针对性练习。

### 业务背景
- 用户需要针对性练习薄弱环节
- 系统根据错题数据分析薄弱知识点
- 自动生成个性化练习试卷
- 提升学习效率和正确率

### 功能目标
- [ ] 智能组卷入口页面
- [ ] 薄弱知识点分析展示
- [ ] 组卷参数配置（难度、题型、数量）
- [ ] 自动组卷算法
- [ ] 练习页面（与现有练习模式复用）

### 约束条件
- 使用Element Plus组件
- 遵循项目设计规范
- 支持响应式布局
- 复用现有练习页面逻辑

## 方案设计

### 页面结构

```
pages/exam/smart/
├── index.vue              # 智能组卷入口页面
├── components/
│   ├── WeakPointsCard.vue # 薄弱知识点卡片
│   ├── ConfigPanel.vue    # 组卷配置面板
│   └── SmartResult.vue    # 组卷结果展示
└── composables/
    └── useSmartExam.ts    # 智能组卷逻辑
```

### 薄弱知识点数据结构

| 字段 | 类型 | 说明 |
|------|------|------|
| knowledgePointId | number | 知识点ID |
| knowledgePointName | string | 知识点名称 |
| parentName | string | 父级分类名称 |
| wrongCount | number | 错题数量 |
| totalCount | number | 总做题数 |
| accuracy | number | 正确率（0-100） |
| weakLevel | 'high' \| 'medium' \| 'low' | 薄弱程度 |

### 组卷参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| questionCount | number | 20 | 题目数量（10/20/30/50） |
| difficulty | number[] | [1,2,3] | 难度筛选（1=简单,2=中等,3=困难） |
| questionTypes | number[] | [1,2,3,4] | 题型筛选（1=单选,2=多选,3=判断,4=填空） |
| weakLevel | string[] | ['high','medium'] | 薄弱程度筛选 |

## 注意事项

### 技术要点
- 薄弱知识点按正确率排序（正确率越低越薄弱）
- 正确率 < 50% 为"high"薄弱
- 正确率 50%-70% 为"medium"薄弱
- 正确率 > 70% 为"low"薄弱

### 已知问题
- 无

### 性能考量
- 薄弱知识点数据量不大，前端处理即可
- 组卷结果可缓存，避免重复请求
