## 技术实现

### 类型定义

```typescript
// 公告类型枚举
enum AnnouncementTypeEnum {
  SYSTEM = 1,    // 系统公告
  ACTIVITY = 2,  // 活动通知
  UPDATE = 3,    // 更新日志
  OTHER = 4      // 其他
}

// 公告VO
interface AnnouncementVO {
  id: number
  title: string
  content: string
  type: number
  status: number
  isTop: boolean
  viewCount: number
  publishTime: number
  // ...
}
```

### API接口

| 接口 | 方法 | 路径 | 描述 |
|------|------|------|------|
| 公告列表 | GET | /member/announcement/page | 分页获取公告 |
| 公告详情 | GET | /member/announcement/get | 获取公告详情 |
| 首页公告 | GET | /member/announcement/home-list | 获取置顶+最近7天公告 |
| 增加浏览量 | GET | /member/announcement/view | 浏览量+1 |

### 组件结构

```
app/

```
