## 需求描述

实现用户学习打卡功能，记录每日学习状态，展示连续打卡天数和打卡日历。

### 业务背景
用户需要激励性的学习机制，通过每日打卡培养学习习惯，同时可以查看自己的学习连续性和历史记录。

### 功能目标
- [ ] 每日打卡功能（点击打卡、获取奖励）
- [ ] 连续打卡天数展示
- [ ] 打卡日历视图（可查看历史打卡记录）
- [ ] 打卡提醒功能（可选）
- [ ] 打卡排行榜（可选扩展）

### 约束条件
- 使用 Nuxt 4 + Vue 3 + TypeScript + Element Plus
- 遵循项目现有代码规范
- 需要后端接口支持（打卡记录、连续天数统计）
