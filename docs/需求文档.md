# 学次元在线题库 - 需求文档

## 1. 项目概述

### 1.1 项目信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 学次元在线题库 (xcy-qthy) |
| 项目类型 | 在线教育题库平台 |
| 目标用户 | 学生、考生、教育机构 |
| 技术栈 | Nuxt 4 + Vue 3 + TypeScript + Element Plus + UnoCSS |

### 1.2 项目目标

打造一个功能完善的在线题库平台，支持：
- 丰富的题库资源浏览和搜索
- 在线练习和模拟考试
- 用户个人中心管理
- 订单和支付功能
- 推广联盟系统

---

## 2. 技术架构

### 2.1 前端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Nuxt | 4.x | 全栈框架 |
| Vue | 3.x | UI框架 |
| TypeScript | 5.x | 类型系统 |

| @nuxtjs/i18n | 最新版 | 国际化 |

### 2.2 项目目录结构

```
app/
├── api/                    # API接口定义
│   ├── user/              # 用户相关接口
```

## 5. 接口定义概览

### 5.1 用户相关接口

| 接口 | 方法 | 路径 | 描述 |
|------|------|------|------|
| 获取用户信息 | GET | `/member/user/get` | 获取当前登录用户信息 |
| 更新用户资料 | PUT | `/member/user/update` | 更新用户资料 |
| 上传头像 | POST | `/member/user/update-avatar` | 上传用户头像 |

### 5.2 题库相关接口

| 接口 | 方法 | 路径 | 描述 |
|------|------|------|------|
| 题库列表 | GET | `/member/qbank/list` | 获取题库列表 |
| 题库详情 | GET | `/member/qbank/get` | 获取题库详情 |
| 题目列表 | GET | `/member/question/list` | 获取题目列表 |

### 5.3 订单相关接口

| 接口 | 方法 | 路径 | 描述 |
|------|------|------|------|
| 创建订单 | POST | `/member/order/create` | 创建新订单 |
| 订单列表 | GET | `/member/order/page` | 分页查询订单 |
| 订单详情 | GET | `/member/order/get` | 获取订单详情 |
