# 题库题目评论 - 前端接口文档（APP客户端）

> **文档类型**：前端对接文档
> **适用端**：APP客户端（UniApp/小程序/APP）
> **接口前缀**：`/app-api`
> **鉴权方式**：JWT Token（Header: `Authorization: Bearer {token}`）

---

## 一、接口概览

| 序号 | 接口功能 | 请求方式 | 接口路径 | 鉴权 |
|-----|---------|---------|---------|------|
| 1 | 获取题目评论列表 | GET | `/app-api/cms/question-comment/page` | 是 |
| 2 | 提交题目评论 | POST | `/app-api/cms/question-comment/submit` | 是 |
| 3 | 获取评论回复列表 | GET | `/app-api/cms/question-comment/reply/{commentId}` | 是 |
| 4 | 提交评论回复 | POST | `/app-api/cms/question-comment/reply/submit` | 是 |
| 5 | 删除我的评论 | DELETE | `/app-api/cms/question-comment/{commentId}` | 是 |
| 6 | 点赞/取消点赞评论 | POST | `/app-api/cms/question-comment/like` | 是 |

```http
Content-Type: application/json
```

```typescript
// 统一响应结构
interface ApiResponse<T> {

  data: T;           // 业务数据
}

// 分页数据结构
interface PageResult<T> {
  list: T[];         // 数据列表
```

| code | 说明 |
|-----|------|
| 0 | 请求成功 |

### 3.1 获取题目评论列表

**接口说明**：获取指定题目的评论列表（分页），包含热门回复

**请求信息**
```http
GET /app-api/cms/question-comment/page
```

**请求参数（Query）**

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|-------|------|-----|------|------|
| questionId | number | 是 | 题目ID | 123 |
| pageNum | number | 否 | 页码，默认1 | 1 |
| pageSize | number | 否 | 每页条数，默认10，最大50 | 10 |

**请求示例**
```http
GET /app-api/cms/question-comment/page?questionId=123&pageNum=1&pageSize=10
```

### 3.2 提交题目评论

**接口说明**：对指定题目发表评论

**请求信息**
```http
POST /app-api/cms/question-comment/submit
```

**请求参数（Body）**

| 参数名 | 类型 | 必填 | 说明 | 限制 |
|-------|------|-----|------|------|
| questionId | number | 是 | 题目ID | - |
| content | string | 是 | 评论内容 | 1-500字 |
| media | string[] | 否 | 图片URL列表 | 最多3张，通过文件上传接口获取URL |

**请求示例**
```json
{
  "questionId": 123,
  "content": "这道题解析不够详细，建议补充",
  "media": [
    "https://xxx.com/image1.jpg",
    "https://xxx.com/image2.jpg"
  ]
}
```

**响应数据**
```typescript
interface SubmitCommentResult {
  success: boolean;    // 是否成功
  commentId?: number;  // 评论ID（成功时返回）
}
```

**响应示例**
```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "success": true,
    "commentId": 1002
  }
}
```

**错误码**

| code | msg | 说明 |
|-----|-----|------|
| 400 | 评论内容不能为空 | content为空 |
| 400 | 评论内容最多500字 | 内容超长 |
| 400 | 最多上传3张图片 | media超过3张 |
| 404 | 题目不存在 | questionId无效 |
| 429 | 操作过于频繁，请稍后再试 | 频率限制 |

## 四、前端实现参考

### 4.1 API封装（uni-app示例）

```typescript
// api/question-comment.ts
import request from '@/utils/request';

export const questionCommentApi = {
  /**
   * 获取题目评论列表
   */
  getPage(params: {
    questionId: number;
    pageNum?: number;
    pageSize?: number;
  }) {
    return request.get('/app-api/cms/question-comment/page', { params });
  },

  /**
   * 提交评论
   */
  submit(data: {
    questionId: number;
    content: string;
    media?: string[];
  }) {
    return request.post('/app-api/cms/question-comment/submit', data);
  },

  /**
   * 获取回复列表
   */
  getReplies(commentId: number, params?: {
    pageNum?: number;
    pageSize?: number;
  }) {
    return request.get(`/app-api/cms/question-comment/reply/${commentId}`, { params });
  },

  /**
   * 提交回复
   */
  submitReply(data: {
    commentId: number;
    toUid: number;
    content: string;
  }) {
    return request.post('/app-api/cms/question-comment/reply/submit', data);
  },

  /**
   * 删除评论
   */
  delete(commentId: number) {
    return request.delete(`/app-api/cms/question-comment/${commentId}`);
  },

  /**
   * 点赞操作
   */
  like(data: {
    commentId: number;
    type: 1 | 2 | 3 | 4;
  }) {
    return request.post('/app-api/cms/question-comment/like', data);
  }
};
```

### 4.3 组件调用示例

```vue
<!-- QuestionComment.vue -->
<script setup lang="ts">
import { ref, onMounted } from 'vue';
import { questionCommentApi } from '@/api/question-comment';
import type { QuestionCommentItem } from '@/types/question-comment';

const props = defineProps<{
  questionId: number;
}>();

const commentList = ref<QuestionCommentItem[]>([]);
const loading = ref(false);
const pageNum = ref(1);
const hasMore = ref(true);

// 加载评论列表
async function loadComments() {
  if (loading.value || !hasMore.value) return;
  loading.value = true;

  try {
    const res = await questionCommentApi.getPage({
      questionId: props.questionId,
      pageNum: pageNum.value,
      pageSize: 10
    });

    if (res.code === 0) {
      const { list, total } = res.data;
      commentList.value.push(...list);
      hasMore.value = commentList.value.length < total;
      pageNum.value++;
    }
  } finally {
    loading.value = false;
  }
}

// 提交评论
async function submitComment(content: string, media: string[]) {
  const res = await questionCommentApi.submit({
    questionId: props.questionId,
    content,
    media
  });

  if (res.code === 0) {
    uni.showToast({ title: '评论成功' });
    // 刷新列表
    commentList.value = [];
    pageNum.value = 1;
    hasMore.value = true;
    await loadComments();
  }
}

// 点赞操作
async function handleLike(commentId: number, currentStatus: boolean) {
  const type = currentStatus ? 2 : 1; // 已点赞则取消，否则点赞
  const res = await questionCommentApi.like({ commentId, type });

  if (res.code === 0) {
    // 更新本地状态
    const comment = commentList.value.find(c => c.id === commentId);
    if (comment) {
      comment.likedByCurrentUser = !currentStatus;
      comment.likeCount = res.data.likeCount;
    }
  }
}

onMounted(() => {
  loadComments();
});
</script>
```

## 五、注意事项

### 5.1 频率限制

| 接口 | 限制 | 说明 |
|-----|------|------|
| 提交评论 | 5次/分钟 | 防止刷评论 |
| 提交回复 | 10次/分钟 | 防止刷回复 |
| 点赞操作 | 20次/分钟 | 防止刷赞 |

### 5.2 评论状态说明

评论提交后会经过以下状态：
- `WAIT_AUDIT(0)` - 待审核（人工审核）
- `PASSED(1)` - 已通过
- `AUTO_PASSED(11)` - 自动通过（无需审核）
- `REFUSE(2)` - 已拒绝

**前端展示逻辑**：只显示状态为 `PASSED` 或 `AUTO_PASSED` 的评论。

### 5.3 图片上传流程

1. 调用通用文件上传接口 `/app-api/infra/file/upload`
2. 获取返回的图片URL
3. 将URL放入 `media` 数组提交评论

### 5.4 敏感词过滤

评论内容会自动进行敏感词检测：
- 包含敏感词 → 进入人工审核（WAIT_AUDIT）
- 无敏感词 → 自动通过（AUTO_PASSED）
