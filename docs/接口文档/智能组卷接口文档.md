# 智能组卷接口文档

## 模块说明

智能组卷模块根据用户的能力评估数据，自动生成针对性的练习试卷。

**模块标识**: `smart-exam`
**基础路径**: `/app-api/member/exam/smart`
**版本**: v1.0
**最后更新**: 2026-03-23

---

## 接口列表

### 1. 获取薄弱知识点列表

**接口说明**: 获取用户的薄弱知识点列表，支持按科目筛选

**请求信息**:
- **HTTP Method**: GET
- **接口路径**: `/member/exam/smart/weak-points`
- **权限要求**: 需登录

**请求参数**:

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| subjectId | number | 否 | - | 科目ID，不传则返回全部 |
| limit | number | 否 | 10 | 返回数量限制 |

**响应参数**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| list | WeakKnowledgePoint[] | 薄弱知识点列表 |

**WeakKnowledgePoint 结构**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | number | 知识点ID |
| name | string | 知识点名称 |
| subjectId | number | 科目ID |
| subjectName | string | 科目名称 |
| accuracy | number | 正确率（0-100） |
| priority | number | 优先级 1-5，5为最高 |
| totalQuestions | number | 该知识点总题数 |
| completedQuestions | number | 已做题数 |
| wrongQuestions | number | 错题数 |

**响应示例**:

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "name": "高等数学-极限",
        "subjectId": 1,
        "subjectName": "高等数学",
        "accuracy": 45,
        "priority": 5,
        "totalQuestions": 50,
        "completedQuestions": 20,
        "wrongQuestions": 11
      }
    ]
  }
}
```

### 3. 生成智能试卷

**接口说明**: 根据智能组卷配置生成试卷

**请求信息**:
- **HTTP Method**: POST
- **接口路径**: `/member/exam/smart/generate`
- **权限要求**: 需登录

**请求体**:

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| qbankId | number | 否 | 题库ID |
| subjectId | number | 否 | 科目ID |
| knowledgePointIds | number[] | 否 | 选择的知识点ID列表（手动模式下必填） |
| difficulty | number | 否 | 难度等级 1-5，默认3 |
| questionCount | number | 是 | 题目数量 |
| durationMinutes | number | 是 | 考试时长（分钟） |
| mode | string | 是 | 组卷模式：auto-自动推荐 manual-手动选择 |
| strategy | string | 否 | 优先策略：weak-薄弱优先 random-随机 mixed-混合 |

**响应参数**:

| 字段名 | 类型 | 说明 |

## 业务逻辑说明

### 薄弱知识点计算

薄弱知识点根据以下条件筛选和排序：

1. **正确率**：正确率 < 60% 的知识点视为薄弱
2. **优先级计算**：
   - 正确率 < 40%：优先级 = 5（紧急）
   - 40% <= 正确率 < 50%：优先级 = 4（重要）
   - 50% <= 正确率 < 60%：优先级 = 3（建议）
   - 60% <= 正确率 < 70%：优先级 = 2（可选）
   - 正确率 >= 70%：不视为薄弱知识点
3. **错题数**：错题数越多，优先级越高

### 智能推荐参数计算

根据薄弱知识点数据自动计算：

1. **推荐题目数**：`min(薄弱知识点数 * 5, 50)`
2. **推荐时长**：`推荐题目数 * 2`（分钟）
3. **建议难度**：根据平均正确率计算
   - 平均正确率 < 40%：难度 = 2
   - 40% <= 平均正确率 < 60%：难度 = 3
   - 平均正确率 >= 60%：难度 = 4

### 组卷策略

1. **薄弱优先**：薄弱知识点题目占比 70%，其他知识点占比 30%
2. **随机抽题**：所有知识点平均分配题目
3. **混合模式**：薄弱知识点题目占比 50%，其他知识点占比 50%

## 与现有接口的关系

智能组卷生成的试卷复用现有考试系统：

1. 生成的试卷ID可用于：
   - 调用 `/member/exam/paper/submit` 提交试卷
   - 调用 `/member/exam/paper/report` 获取考试报告
2. 考试历史记录与普通模拟考试记录统一存储
