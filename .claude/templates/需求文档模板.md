**跨端渲染模式**：
- **SSR**：服务端渲染，SEO友好，首屏快速
- **SSG**：静态生成，CDN部署，适合内容站点
- **SPA**：单页应用，适合后台管理系统
- **Hybrid**：混合模式，按需选择渲染策略

**多端适配**：
- PC端：桌面端完整功能
- 移动端：响应式布局/触摸优化
- PWA：离线访问/添加到主屏幕

**跨端渲染配置示例**：
```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  // 路由渲染规则
  routeRules: {
    // 首页SSR渲染
    '/': { prerender: true },
    // 后台管理SPA模式
    '/admin/**': { ssr: false },
    // API接口
    '/api/**': { cors: true }
  },

  // PWA配置
  pwa: {
    registerType: 'autoUpdate',
    manifest: {
      name: 'My App',
      short_name: 'App',
      theme_color: '#ffffff'
    }
  },

  // 模块配置
  modules: [
    '@vite-pwa/nuxt',        // PWA支持
    '@nuxtjs/device',        // 设备检测
    '@nuxtjs/i18n',          // 国际化
    '@unocss/nuxt'           // 原子化CSS
  ]
})
```
